About Me

            안녕하세요. 신입 개발자 이동엽 입니다.

            웹 개발자가 되기 위해 고등학교와 대학교에서 교육을 받으며, 
            두 번의 교육원 수료를 통해 프로젝트 경험을 쌓았습니다.

            더 나은 개발자로 성장하기 위해 Back-End와 Front-End 
            모두에 관심을 가지고 공부하고 있습니다.

            Back-End 개발에서는 주로 Java 언어를 사용하며, 
            Spring Framework를 활용해 4번의 프로젝트를 진행한 경험이 있습니다.

            Front-End 개발은 최신 트렌드를 반영하여, 
            React 라이브러리를 활용한 프로젝트를 진행한 경험이 있습니다.

            현재는 Node.js, Nest.js, TypeScript를 공부하고 있으며,
            크로스 플랫폼 개발에도 큰 관심을 두고 있어 React Native와 
            Flutter를 활용한 모바일 개발 공부도 계획 중입니다.
